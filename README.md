## 简体中文

一款轻量、干净且功能强大的 YouTube 双语字幕扩展。支持智能翻译、深浅色主题切换以及高度自定义的字幕样式配置。

### ✨ 特性
- **双语对齐**：同时显示视频原文字幕与翻译字幕，语言学习与观影利器。
- **高度自定义**：自由调节字体、字号、颜色，甚至全屏模式下的背景透明度。
- **原生样式融合**：经过精心设计的 UI 卡片，完美融入 YouTube 原生播放器界面。
- **三语界面**：插件面板原生支持中文、英文和西班牙语无缝切换。

### 🌐 兼容性
本插件基于 Chrome Extension Manifest V3 标准开发，完美兼容所有基于 Chromium 内核的浏览器：
- 🌐 **Google Chrome**
- 🔷 **Microsoft Edge**
- 🦁 **Brave Browser**
- 🔴 **Opera / Vivaldi** 等

### 🛠️ 安装方法

#### 方法一：从 Chrome 应用商店安装（推荐）
1. 使用上述任意基于 Chromium 的浏览器访问 [Chrome Web Store (即将上线)](#)。 *(注：项目已提交审核，通过后将第一时间更新下载链接)*
2. **Edge 用户提示**：如果是第一次在 Edge 中安装 Chrome 商店的插件，请点击页面顶部横幅的 **“允许来自其他商店的扩展”**，然后点击 **“添加至 Chrome”** 即可直接无缝安装。

#### 方法二：开发者模式安装（本地调试 / GitHub 源码下载）
1. 点击本页面右上角的 `Code` -> `Download ZIP` 下载源码并解压到本地。
2. 打开浏览器，进入扩展程序管理页面（可在地址栏直接输入）：
   - **Chrome**: `chrome://extensions/`
   - **Edge**: `edge://extensions/`
3. 勾选并开启页面右上角的 **“开发者模式”**。
4. 点击左上角的 **“加载已解压的扩展程序”** 按钮。
5. 选择你解压的**项目根目录**（包含 `manifest.json` 的那个文件夹），即可成功加载！

### ⌨️ 快捷键
- **Alt + S** (Mac 上为 **Option + S**)：在全屏模式下快速呼出/隐藏字幕样式设置面板。

## English

A lightweight, clean, and powerful YouTube dual subtitles extension. It features smart translation, light/dark theme switching, and highly customizable subtitle styles.

### ✨ Features
- **Dual Subtitles**: Displays both original and translated subtitles simultaneously—perfect for language learners and international viewers.
- **Highly Customizable**: Freely adjust fonts, font sizes, colors, and even background opacity for fullscreen mode.
- **Native Integration**: Meticulously designed UI cards that blend seamlessly into the native YouTube player.
- **Trilingual UI**: The extension popup natively supports seamless switching between English, Chinese, and Spanish.

### 🌐 Compatibility
This extension is built on the Chrome Extension Manifest V3 standard and is 100% compatible with all Chromium-based browsers:
- 🌐 **Google Chrome**
- 🔷 **Microsoft Edge**
- 🦁 **Brave Browser**
- 🔴 **Opera / Vivaldi**, etc.

### 🛠️ Installation

#### Method 1: Install from Chrome Web Store (Recommended)
1. Visit the [Chrome Web Store (Coming Soon)](#) using any Chromium-based browser. *(Note: The extension has been submitted for review and the link will be updated immediately upon approval)*
2. **Note for Edge users**: If this is your first time installing an extension from the Chrome Web Store, click **"Allow extensions from other stores"** on the banner at the top of the page, then click **"Add to Chrome"** to install seamlessly.

#### Method 2: Developer Mode Installation (Local Debugging / GitHub Download)
1. Click `Code` -> `Download ZIP` at the top right of this page to download and extract the source code locally.
2. Open your browser and navigate to the extension management page:
   - **Chrome**: Enter `chrome://extensions/` in the address bar.
   - **Edge**: Enter `edge://extensions/` in the address bar.
3. Enable **"Developer mode"** via the toggle switch in the top right.
4. Click the **"Load unpacked"** button in the top left.
5. Select the **root folder** of the extracted project (the one containing `manifest.json`), and the extension will be successfully loaded!

### ⌨️ Shortcuts
- **Alt + S** (**Option + S** on Mac): Toggle the subtitle settings panel quickly while in fullscreen mode.

## Español

Una extensión ligera, limpia y potente de subtítulos dobles para YouTube. Cuenta con traducción inteligente, cambio de tema claro/oscuro y estilos de subtítulos altamente personalizables.

### ✨ Características
- **Subtítulos Dobles**: Muestra los subtítulos originales y traducidos simultáneamente, ideal para estudiantes de idiomas y espectadores internacionales.
- **Altamente Personalizable**: Ajusta libremente las fuentes, tamaños de letra, colores e incluso la opacidad del fondo en el modo de pantalla completa.
- **Integración Nativa**: Diseñado meticulosamente con tarjetas de interfaz de usuario que se integran a la perfección con el reproductor nativo de YouTube.
- **Interfaz Trilingüe**: El panel de la extensión admite el cambio fluido entre español, inglés y chino.

### 🌐 Compatibilidad
Esta extensión está desarrollada bajo el estándar Chrome Extension Manifest V3 y es 100% compatible con todos los navegadores basados en Chromium:
- 🌐 **Google Chrome**
- 🔷 **Microsoft Edge**
- 🦁 **Brave Browser**
- 🔴 **Opera / Vivaldi**, etc.

### 🛠️ Instalación

#### Método 1: Instalar desde Chrome Web Store (Recomendado)
1. Visita el enlace de [Chrome Web Store (Próximamente)](#) utilizando cualquiera de los navegadores basados en Chromium mencionados anteriormente. *(Nota: El proyecto ha sido enviado a revisión y el enlace se actualizará inmediatamente después de su aprobación)*
2. **Nota para usuarios de Edge**: Si es la primera vez que instalas una extensión desde la Chrome Web Store, haz clic en **"Permitir extensiones de otras tiendas"** en el banner superior de la página, y luego haz clic en **"Añadir a Chrome"** para instalarla sin problemas.

#### Método 2: Instalación en Modo de Desarrollador (Depuración Local / Descarga desde GitHub)
1. Haz clic en `Code` -> `Download ZIP` en la esquina superior derecha de esta página para descargar y extraer el código fuente en tu ordenador.
2. Abre tu navegador e introduce la siguiente dirección en la barra de navegación:
   - **Chrome**: Escribe `chrome://extensions/`
   - **Edge**: Escribe `edge://extensions/`
3. Activa el interruptor de **"Modo de desarrollador"** (Developer mode) en la esquina superior derecha.
4. Haz clic en el botón **"Cargar descomprimida"** (Load unpacked) en la esquina superior izquierda.
5. Selecciona la **carpeta raíz** del proyecto extraído (la que contiene el archivo `manifest.json`), ¡y la extensión se cargará correctamente!

### ⌨️ Atajos de Teclado
- **Alt + S** (**Option + S** en Mac): Muestra u oculta rápidamente el panel de configuración de subtítulos en el modo de pantalla completa.
